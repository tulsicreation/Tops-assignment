Website: A collection of web pages that are grouped together and usually connected by hyperlinks. A website is identified by a domain name and hosted on a web server. For example, MDN Web Docs is a website that provides learning resources for web development. Webpage: A document that can be displayed in a web browser and contains HTML code and other elements, such as images, CSS stylesheets, JavaScript files, etc. A webpage is accessed by a URL and can be part of a website or a standalone document. For example, this page is a webpage that explains what a webpage is. Web browser: A software application that allows users to access and view webpages on the Internet. A web browser can interpret HTML, CSS, and JavaScript code and render the webpage accordingly. Some common web browsers are Firefox, Chrome, Edge, Safari, and Opera. Web server: A computer or a software program that hosts websites and responds to requests from web browsers. A web server can store and deliver static or dynamic webpages to the browser using the HTTP protocol. Some common web server software are Apache, Nginx, IIS, and Tomcat. HTML: HyperText Markup Language, the standard markup language for creating webpages. HTML consists of tags that define the structure and content of a webpage, such as headings, paragraphs, links, images, etc. HTML files have the extension .html or .htm. CSS: Cascading Style Sheets, a style sheet language that is used to describe how HTML elements are displayed on a webpage. CSS can control the layout, colors, fonts, animations, etc. of a webpage. CSS files have the extension .css.
